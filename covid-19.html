COVID-19 Info

This was a project I started back in July 2020 when I first learned React.Js. I was the sole creator/developer for the first year of the project before I was able to hire help.

The system shows all the data according to total death, total infected, total recovered in Covid-19 throughout the world.

Data are shown in graphical way also for easy visualisation purpose.

In case of India only user can even select different states of India and, be aware of it's condition.

Technologies Used:
  • - React JS
  • - Chart Js
  • - Css
  • - Netlify
  • - Git & Github
  • - covid19.mathdro.id/api
